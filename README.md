This project presents an interactive Tableau dashboard designed for business stakeholders to monitor key performance  identify trends, risks, and opportunities.
The dashboard uses a sales dataset (Superstore-style) and focuses on delivering clear insights into Sales, Profit, and Growth through intuitive visuals and interactivity.

Objective :
Track overall business performance
Analyze trends over time
Compare regional and category-level performance
Enable stakeholders to explore data dynamically

Tools Used :
Tableau Public
Microsoft Excel / CSV Dataset
